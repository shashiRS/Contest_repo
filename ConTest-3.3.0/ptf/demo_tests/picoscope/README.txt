This is the location where test cases can be written.
Following is some important information:

 1. setup.pytest file contains setup and teardown function for setting up your test cases
 2. A sample test file has been created 'swt_sample_test.pytest'
 3. In this folder you can create many swt_<what_ever>.pytest files also in sub-folders
 4. You can create .py files and import them as normal scripts in .pytest files
 5. Read PTF documentation 'Help/User Manual' for detail information
